## Детали реализации

### 1. Конфигурация (`app/config.py`)

Класс `Config` содержит:

| Параметр | Значение |
|---|---|
| `SECRET_KEY` | env `SECRET_KEY`, по умолчанию `dev-secret-key-change-in-production` |
| `SQLALCHEMY_DATABASE_URI` | SQLite (`instance/testing.db`) |
| `SQLALCHEMY_TRACK_MODIFICATIONS` | `False` |
| `DEFAULT_TIME_LIMIT` | 1 (секунда) |
| `DEFAULT_MEMORY_LIMIT` | 256 (КБ) |
| `PYTHON_EXEC` | env `PYTHON_EXEC`, по умолчанию `python` |
| `CPP_COMPILER` | env `CPP_COMPILER`, по умолчанию `g++` |
| `PASSWORD_CHARS` | строчные буквы, цифры, спецсимволы |
| `PASSWORD_LENGTH` | 6 |
| `TEACHER_USERNAME` | env, по умолчанию `admin` |
| `TEACHER_PASSWORD` | env, по умолчанию `admin123` |

Класс `TestConfig` наследуется от `Config`:
- `TESTING = True`
- `SQLALCHEMY_DATABASE_URI = 'sqlite:///:memory:'`
- `WTF_CSRF_ENABLED = False`

### 2. Фабрика приложения (`app/__init__.py`)

Функция `create_app(config_class=Config)`:

1. Создаёт Flask-приложение и загружает конфигурацию.
2. Создаёт директорию `uploads/`, если её нет.
3. Инициализирует SQLAlchemy (`db.init_app(app)`).
4. Регистрирует 3 blueprints: `auth`, `teacher`, `student`.
5. Добавляет маршрут `/` → `base.html`.

### 3. Заглушки blueprints

| Blueprint | Файл | URL-префикс |
|---|---|---|
| `auth` | `app/auth.py` | `/` |
| `teacher` | `app/teacher.py` | `/teacher` |
| `student` | `app/student.py` | `/` |
| `checker` | `app/checker.py` | — |

### 4. Модель `Teacher` (`app/models.py`)

```python
class Teacher(db.Model):
    __tablename__ = 'teacher'
    id = db.Column(db.Integer, primary_key=True)
    username = db.Column(db.String(80), unique=True, nullable=False)
    password_hash = db.Column(db.String(256), nullable=False)
```

Создана как заглушка, необходимая для работы `init_db.py`. Полные модели будут реализованы в подзадаче 2.

### 5. Базовый шаблон (`app/templates/base.html`)

HTML5-шаблон с:
- Блоками `{% block title %}`, `{% block content %}`, `{% block scripts %}`
- Подключением `style.css` и `main.js`

### 6. Стили (`app/static/css/style.css`)

- Переменные CSS: белый фон, синий акцент (`#2c5aa0`), контрастный шрифт
- Базовые классы: `.container`, таблицы, формы, кнопки, сообщения

### 7. Точка входа (`run.py`)

```python
from app import create_app
app = create_app()
if __name__ == '__main__':
    app.run(debug=True)
```

### 8. Инициализация БД (`init_db.py`)

- Создаёт директорию `instance/`
- Вызывает `db.create_all()`
- Создаёт единственную запись `Teacher` из конфигурации, если её нет

### 9. Зависимости (`requirements.txt`)

```
flask
flask-sqlalchemy
openpyxl
pytest
pytest-flask
```

Установлены в виртуальное окружение `venv/` (Python 3.12.7).
