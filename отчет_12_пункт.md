## 1. Что реализовано

### 1.1. Конфигурация развёртывания (раздел 12 ТЗ)

#### `instance/config.example.py` — шаблон конфигурации сервера

Создан файл-шаблон, который копируется в `instance/config.py` и редактируется под конкретный сервер. Содержит:

| Параметр | Описание |
|---|---|
| `SECRET_KEY` | Секретный ключ для подписи сессий (инструкция по генерации в комментариях) |
| `SQLALCHEMY_DATABASE_URI` | Абсолютный путь к базе данных SQLite |
| `DEFAULT_TIME_LIMIT` / `DEFAULT_MEMORY_LIMIT` | Лимиты по умолчанию для задач |
| `PYTHON_EXEC` / `CPP_COMPILER` | Пути к интерпретаторам и компиляторам |
| `TEACHER_USERNAME` / `TEACHER_PASSWORD` | Учётные данные учителя (для `init_db.py`) |
| `CSRF_PROTECTION_ENABLED` | Включение CSRF-защиты (рекомендуется `True` в production) |

#### Поддержка `instance/config.py` в `create_app()` (`app/__init__.py`)

```python
def create_app(config_class=Config):
    instance_path = os.path.join(os.path.dirname(os.path.dirname(__file__)),
                                 'instance')
    app = Flask(__name__, instance_path=instance_path,
                instance_relative_config=True)
    app.config.from_object(config_class)
    app.config.from_pyfile('config.py', silent=True)
```

`from_pyfile('config.py', silent=True)` загружает конфигурацию из `instance/config.py`, если файл существует, и перекрывает значения по умолчанию из `app/config.py`. Если файл отсутствует — используются значения по умолчанию.

### 1.3. WSGI-сервер (раздел 12 ТЗ)

#### `wsgi.py` — точка входа для production

```python
from app import create_app

app = create_app()

if __name__ == '__main__':
    from waitress import serve
    serve(app, host='0.0.0.0', port=8000)
```

| Платформа | Команда | Порт |
|---|---|---|
| Windows | `python wsgi.py` (waitress) | 8000 |
| Linux | `gunicorn -w 4 -b 0.0.0.0:8000 wsgi:app` | 8000 |
